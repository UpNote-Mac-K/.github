## Overview

UpNote Advanced is a clean and fast note-taking application that organizes writing into notebooks with rich text editing and reliable cross-device sync, built around speed and simplicity rather than feature accumulation.

The notebook system nests notes into a clear hierarchy, letting you structure information in a way that mirrors how you actually think about different areas of work and life.

Rich text formatting supports headings, lists, tables, and images without the loading delays or processing pauses that heavier note applications introduce as a library grows.

End-to-end encryption protects notebooks containing sensitive personal or financial information, applying security at the notebook level without requiring a separate locked application.

A built-in note linking feature connects related notes directly within the text, letting a personal knowledge base grow organically as new notes reference earlier ones.

Cross-platform availability on Mac, Windows, iOS, and Android keeps the same notebook library accessible regardless of which device is at hand, with sync handled automatically in the background.

For users who want a note app that stays fast and pleasant to use even as their note library grows into the thousands, UpNote Advanced is the notebook built around that lasting responsiveness.

## Performance

UpNote balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
